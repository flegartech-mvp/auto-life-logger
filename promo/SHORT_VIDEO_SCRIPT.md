# SHORT VIDEO SCRIPT — AutoLifeLogger

**Format:** 30-second screen recording of Chrome browser, popup and dashboard visible
**Tone:** Direct, slightly uncomfortable (the truth about screen time should sting a little)

---

## Hook — 0 to 5 seconds

**Screen shows:** A Chrome browser with 6-8 tabs open — mix of GitHub, YouTube, Twitter, VS Code Web, Reddit, Gmail. The AutoLifeLogger extension icon in the toolbar shows a badge with the number "54" in orange (below 70 = struggling).

**Caption overlay:** "Your browser knows where your time went. This extension just makes you look at it."

---

## Show — 5 to 20 seconds

**Screen shows (in sequence):**

1. Click the toolbar icon — popup opens showing: Today's productive time: 1h 10m | Distracting: 1h 05m | Discipline score: 54 (displayed in orange). A red "Distraction Loop Detected" badge is visible.

2. Click "Top Sites" within the popup — ranked list appears:
   - YouTube — 42 min (red, distracting)
   - GitHub — 38 min (green, productive)
   - Reddit — 23 min (red, distracting)
   - Gmail — 14 min (grey, neutral)

3. Click "Weekly Trends" — bar chart shows 7 days. Monday-Wednesday scores are green (75-85). Thursday-Saturday are orange/red (50-60). Sunday is deep red (35). No narration needed — the visual tells the story.

4. Scroll to "Focus Patterns" section — three badges visible: "Distraction Loop x3", "Time Overrun: YouTube", "Micro-distractions: 8 today".

**Caption overlay during Focus Patterns section:** "It knows when you bounced back to YouTube 3 times in one hour."

---

## Close — 20 to 30 seconds

**Screen shows:** User clicks "Enable Strict Mode" toggle in the dashboard. Then navigates to youtube.com — tab immediately redirects to a clean block page: "Strict Mode is active. You blocked this site." The discipline score badge on the toolbar ticks up from 54 to 55 within a few seconds.

**Caption overlay:** "Take back your time. Free. No account. Chrome Web Store."

**Final frame:** Extension icon + name "AutoLifeLogger" + GitHub/store link on a dark background.

---

## Production Notes

- Pre-populate chrome.storage.local with realistic fake session data before recording so charts and scores are meaningful
- Use a clean Chrome profile — no other extensions visible in toolbar
- The discipline score badge colour change (orange to green) is a satisfying visual; try to capture it ticking up after enabling Strict Mode
- Record at 1280×800 — popup and full dashboard should both be visible without zooming
